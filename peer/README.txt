P2P FILE SHARING SYSTEM

Project Description:
This project implements a Peer-to-Peer (P2P) File Sharing System in Python.

Features:
- File chunking
- Parallel download
- Multi-peer download
- Hash verification (SHA-256)
- Resume download support
- File merging

Technologies Used:
- Python
- Socket Programming
- Multithreading
- Hashing (SHA-256)

How to Run:

1. Start upload server:
python test_upload_server.py

2. Run download:
python test_parallel_download.py

3. Verify chunks:
python test_hash_verifier.py

4. Resume missing chunks:
python test_resume_download.py
